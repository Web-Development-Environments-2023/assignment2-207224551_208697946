About the Game:

This is a SpaceShips Game.

For the development we used a template of the canonball Game and created the relevant functionality.

Instructions:

In the CONFIGURATION page choose a key for shooting and game time.

For movement of your spaceships use the keyboard's arows.

Results:

Champion = killed all bad spaceships.

Winer = timeout & score > 100.

Loser = killed by bad spaceships.
